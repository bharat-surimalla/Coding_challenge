1.This is a simple webserver created using node and express.
2.This server sends client the webpage when received a particular request.
3.Make sure node and npm is installed in the system.
4.Download and change directory to the particular folder
5.Create a json package using 'npm init --yes' command.
6.Install express with the command 'npm install --save express' from the command Prompt.
7.Run the code using node index.js command.
8.Open Browser and enter the url to see the output sent by server.
